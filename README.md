## 📖 Overview
**RollingPuppy** is a high-performance **PWA** designed to turn the toxic cycle of social media "scrolling" into a competitive, rewarding "involution" (内卷) experience. 

## 🚀 Features
* **⏱️ Rolling Timer:** Blocks short-video distractions and tracks deep work.
* **💰 Rolling Points (卷分):** Earn currency for every minute you focus.
* **🛍️ Puppy Shop:** Spend points on Lofi music, custom fonts, and UI themes.
* **👥 RollingTeam (內卷小組):** Form groups and out-grind your friends in real-time.
* **🏆 Rollingboard (卷榜):** Compete globally for the "Ultimate JuanDog" title.
